**Интро**

Проект создавался для обучения базовому JS, содержит в себе результаты работы 4го, 5го и 6го спринтов курса веб разработчика от компании Яндекс Практикум.


На странице представлен редактируемый профиль и карточки с возможностью поставить like. Формы валидируются через JS.

**Технологии**

В проекте используются технологии:
1. БЭМ
2. Flex
3. Grid
4. Адаптивная верстка
5. JS (функции, слушатели, метод foreach, работа с массивом)
6. Валидация форм
